# Vrinda Store Sales Analysis 📊

## Objective
The objective of this project is to analyze Vrinda Store sales data using MS Excel and create an interactive dashboard to identify customer behavior, sales trends, and business insights.

---

## Dataset Source
Dataset: Kaggle

---

## Project Roadmap
- Data Cleaning
- Data Processing
- Pivot Tables & Charts
- Dashboard Creation
- Insights Generation

---

## Key Insights
- Women contribute around 65% of total purchases.
- Maharashtra, Karnataka, and Uttar Pradesh are the top-performing states.
- Customers aged 30–49 years contribute nearly 50% of sales.
- Amazon, Flipkart, and Myntra are the highest contributing sales channels.

---

## Conclusion
Target women customers aged 30–49 years in top-performing states through promotions and offers on Amazon, Flipkart, and Myntra.
